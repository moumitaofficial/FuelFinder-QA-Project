# FuelFinder – Manual QA Testing Project

 Project Overview
This project contains manual QA testing documentation for the FuelFinder application.  
It includes test cases for major modules such as login, booking, payment, location, and feedback.

The goal is to ensure system functionality, reliability, and user experience through structured manual testing.

---

##  Modules Covered
- Login Functionality
- Fuel Booking System
- Payment Process
- Location Services
- Search & Filtering
- User Feedback System

---

##  Contents
- Test Case Document (15+ test cases)
- Positive & Negative test scenarios
- Functional testing coverage

---

##  Tools Used
- Manual Testing
- Microsoft Excel / Word
- Basic QA methodology (SDLC/STLC)

---

##  Objective
To validate that the FuelFinder system works correctly under normal and edge conditions and ensures smooth user experience.
